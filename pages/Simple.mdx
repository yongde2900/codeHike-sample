## Annotations
* foucs: 讓目標代碼明亮，並使其他代碼變得黯淡
* mark: 增加背景顏色在目標區塊
* withClass:新增一個class至目標程式碼
* link:為目標程式碼新增連結
* from : 包含來自外部文件的代碼

_`focus=1,3:5,7[1:4,7:9]`_ 我們可以再檔名後面加上這樣一串，用 _`:`_ 來代表連續行數， 用 _`,`_ 來分隔，用 _`[]`_ 來代表該行內的第幾個字母


```markdown tab.mdx
# from ./CH.mdx
```

## CH.Code
在 mdx 檔案中使用 _`<CH.Code>`_ 以 Tab 或 Panels 來展示你的程式碼檔案，也可以針對這部份的程式碼設定 style 或 config
### Tab 
當你在 _`<CH.Code`_ 區塊中使用超過一個以上的程式碼區塊時，就會以 Tab 顯示。
<CH.Code>
```py one.py
# from ./CH.mdx
```
```py two.py
# from ./CH.mdx
```

</CH.Code>
### Panels 
你也可以同時展示兩段不同檔案的程式碼，在兩個 Panels 中，使用 _`---`_ 分割線來分隔兩個 Panels 。
<CH.Code>
```py one.py
# from ./panel.mdx
```
---
```py two.py
# from ./panel.mdx
```

</CH.Code>
